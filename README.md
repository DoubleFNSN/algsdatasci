## Algorithms

### Definition
  - transforms one set of values into a new set of values
  - efficiency in terms of time and space

### Running Time
Definition: number of primitive computational steps performed
For example,

  - arithmetic
    * add
    * subtract
    * multiply
    * divide
  - data movement
    * load
    * store
    * copy
  - control
    * branching
    * subroutine call
    * return
